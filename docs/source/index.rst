Luna
====

Inspired by `Lucid <https://github.com/tensorflow/lucid>`_, **Luna** is a Feature Visualization package for Tensorflow.
While Lucid does not support Tensorflow 2, **Luna** was built with Tensorflow 2 at its core.

You can install Luna via pip as follows: ``pip install luna_fviz``

.. note::

   This project is under active development. You can find the repository on `GitHub <https://github.com/Sparkier/luna>`__.
